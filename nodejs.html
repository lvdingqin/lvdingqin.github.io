# 学习nodejs

# 说明

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。（简单来说就是一个用来执行javascript代码的程序）

Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。

Node.js 的包管理器 npm，是全球最大的开源库生态系统。通过npm可以通过开源库下载非常多的javascript模块，大大的提高了使用javascript开发代码的效率，通过npm库提供的脚手架工具可以快速开发出实际应用项目。

# 安装

访问nodejs官方网站nodejs.org，点击下载8.11.1 LTS版本并按正常软件安装流程安装好

安装好后，打开命令行工具（windows下为cmd，mac下为terminal）

输入node -v查看nodejs安装版本

输入npm -v查看npm安装版本

# 配置npm国内源

npm config set registry https://registry.npm.taobao.org 
npm info underscore （如果上面配置正确这个命令会有字符串response）
