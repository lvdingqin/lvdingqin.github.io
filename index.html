# 指南

本教程主要目标是通过讲解前端开发必须的核心知识，让一个小白从零开始到可以进行前端开发任务为止。这中间包括的知识点主要有html、css、javascript、nodejs、vue、小程序这些知识点。

# html

超文本标记语言（英语：HyperText Markup Language，简称：HTML）是一种用于创建网页的标准标记语言。主要用来标记一个网页上主要有那些内容，以下为一个基础HTML框架。

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>菜鸟教程(runoob.com)</title>
</head>
<body>
 
<h1>我的第一个标题</h1>
 
<p>我的第一个段落。</p>
 
</body>
</html>

# css

css 指层叠样式表 (Cascading Style Sheets)，样式表定义如何显示 HTML 元素。

#param {
    text-align:center;
    color:red;
}
.center {
    text-align:center;
}

# javascript

javascript 是 Web 的编程语言。JavaScript 是互联网上最流行的脚本语言，这门语言可用于 HTML 和 web，更可广泛用于服务器、PC、笔记本电脑、平板电脑和智能手机等设备。

alert("我的第一个 JavaScript");

# nodejs

Node.js 就是运行在服务端的 JavaScript。Node.js是一个事件驱动I/O服务端JavaScript环境，基于Google的V8引擎，V8引擎执行Javascript的速度非常快，性能非常好。

# vue

Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

# 小程序

微信小程序是一种全新的连接用户与服务的方式，它可以在微信内被便捷地获取和传播，同时具有出色的使用体验。小程序本质上参考了VUE框架的很多设计思想，了解VUE基本编写小程序问题不大。

# 小游戏

微信小游戏是小程序的一个类目。用户完成小程序注册后，可选择“游戏”类目并开始开发、调试小游戏。
